## About
A list of all files in the AW3D dataset contained on the Jaxa Servers.

## Usage
1. Create an account on: https://www.eorc.jaxa.jp/ALOS/en/dataset/aw3d30/registration.htm
2. Clone this Repository
3. Run: ``` parallel --gnu -a files.txt wget ```
